# Mir-Lunto 个人空间

一个现代化的个人网站，具有玻璃态设计、评论系统和Supabase后端集成。

## 功能特性

### 🌟 核心功能
- **响应式设计**：适配各种设备屏幕
- **玻璃态UI**：现代化的毛玻璃效果设计
- **动态背景**：视频背景与静态背景自动切换
- **搜索功能**：实时搜索文章和内容

### 💬 评论系统
- **Supabase后端**：安全的数据存储
- **管理员认证**：Supabase Auth集成
- **实时同步**：评论自动同步
- **表情支持**：丰富的表情选择器
- **回复功能**：支持嵌套回复
- **点赞功能**：用户点赞互动

### 🔐 管理员功能
- **安全登录**：Supabase身份验证
- **权限管理**：行级安全策略（RLS）
- **数据统计**：实时数据统计面板
- **批量操作**：导出数据、删除管理
